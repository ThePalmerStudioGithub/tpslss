Backend Codes:

Backend-Started

This code means the backend has started successfully.

Backend-Connection-Test-Started

This code means that a internet connection test has been started.

Backend-Countdown-Executed

This code means that the countdown has been started.

Backend-Refreshed-Log

This code means that the log has been refreshed.

Backend-Output-Manager-Started

This code means that the Output Manager has been started/opened.

RPC Codes:

RPC-Premade-Presence-Set

This code means that a premade presence has been set.

RPC-Initialized

This code means that the RPC has initialized/started up successfully.

Socket Codes:

001-Socket

This code means that the websocket is not open for the premade "Streaming" presence.

002-Socket

This code means that the websocket is not open for the premade "Update to Current Scene" presence.
